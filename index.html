Course Overview

Learning Objectives

  • Understand the Model-View-Template architecture and Django's implementation
  • Design and implement relational database schemas using Django ORM
  • Build RESTful APIs using Django REST Framework
  • Implement authentication, authorization, and security best practices
  • Deploy applications to production environments with CI/CD pipelines

Prerequisites

Python Programming (CS-101)
Web Technologies (HTML/CSS/JS)
Database Systems (CS-302)
Version Control (Git)
